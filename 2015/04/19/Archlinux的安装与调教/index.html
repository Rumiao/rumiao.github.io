Archlinux的安装与调教

發表於 2015-04-18   |   分類於 Linux

安装前的准备

  • 1 启动盘的制作
    找到u盘的设备文件名,比如/dev/sdb1.

    sudo fdisk -l

    格式化u盘.

    sudo umount /dev/sdb1
sudo mkfs.vfat /dev/sdb1

    linux的文件传输命令帮我们制作启动盘,if后是Archlinux镜像文件的路径,of后是u盘你的设备文件名,切记不是/dev/sdb1 而是/dev/sdb.bs是传输缓冲区大小,可以是1M,2M,4M等.

    sudo dd if=Path_of_Archlinux.iso of=/dev/sdb bs=2M

    至此启动盘制作完成

  • 2 UEFI+GPT or BIOS+MBR?
    分清你的电脑采用哪种启动引导方式和硬盘分区是正确安装的关键.现在的很多电脑已经采用了UEFI+GPT了.现在就以UEFI+GPT为例来安装.

开始安装

  • 1 重要
    如果你是UEFI启动,那么u盘启动后会含有UEFI字样的启动选项,从此处进入开始安装系统,否则最终引导会出现问题.进入后选择第一项开始安装.

    Arch Linux archiso x86_64 UEFI USB
UEFI Shell x86_64 v1
UEFI Shell x86_64 v2
EFI Default Loader

  • 2 分区
    如果是一块不含操作系统的硬盘,直接分区就可以.如果要把Arch作为第二个操作系统,那么就应该留出大小合适的硬盘出来.我在win7下预留了100G来装Arch.
    UEFI+GPT分区需要一个 EFI 系统分区.如果是和 Windows 双系统启动,此分区已经存在,不要重新创建.
    | 分区 | 大小 | 文件格式 | 挂载 |
    | ——— | :——-: | ————: | —— |
    | /dev/sda7 | 513M | fat32 | /boot |
    | /dev/sda8 | 40G | ext4 | / |
    | /dev/sda9 | 59G | ext4 | /home |
    我采取了这样的方案.分区会用到fdisk, mkfs, mount, umount这些命令.

  • 3 挂载
    必须先把/挂载在/mnt上,其他目录在/mnt/下创建后再挂载.

    sudo mount /dev/sda8 /mnt/

    然后挂载其他分区, EFI分区挂载在/mnt/boot

    mkdir /mnt/boot
sudo mount /dev/sda7 /mnt/boot
mkdir /mnt/home
sudo mount /dev/sda9 /mnt/home

  • 4 联网

    • 1.无线

      iw dev

      找到字段Interface wlp3s0

      wifi-menu wlp3s0

      连接到网络.

    • 2.有线
      参考Wiki

  • 5 选择安装镜像
    安装前需要编辑 /etc/pacman.d/mirrorlist, 将偏好的镜像放到最前面.

    vi /etc/pacman.d/mirrorlist

    Arch Linux repository mirrorlist
    Sorted by mirror score from mirror status page
    Generated on 2012-MM-DD

    Server = http://mirrors.zju.edu.cn/archlinux/$repo/os/$arch
    Server = http://mirrors.163.com/archlinux/$repo/os/$arch
    …

    sudo pacman -Syy 来刷新镜像列表

  • 6 安装基本系统

    pacstrap -i /mnt base base-devel

  • 7 生成fstab

    genfstab -U -p /mnt >> /mnt/etc/fstab

  • 8 Chroot 并开始配置新系统

    arch-chroot /mnt /bin/bash

    • 1.Locale
      移除对应行前面的注释符号

      vi /etc/locale.gen

      en_US.UTF-8 UTF-8
      zh_CN.UTF-8 UTF-8
      zh_TW.UTF-8 UTF-8

      接着执行locale-gen以生成locale讯息：

      locale-gen

      本地配置,设置任何中文locale,会导致tty乱码.

      echo LANG=en_US.UTF-8 > /etc/locale.conf

    • 2.时区

      ln -s /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

    • 3.硬件时间

      hwclock --systohc --utc

    • 4.配置网络

      pacman -S iw wpa_supplicant dialog

      就可以使用wifi-menu连接无线网了

    • 5.主机名

      echo myhostname > /etc/hostname

      并在/etc/hosts添加同样的主机名：

      127.0.0.1 localhost.localdomain localhost myhostname
      ::1 localhost.localdomain localhost myhostname

    • 6.设置root密码

      passwd

  • 9 安装并配置bootloader
    到了关键的一步,一定要按照wiki上的选择适合自己的,UEFI主板有多种引导方式,推荐GRUB引导.

    sudo pacman -S grub efibootmgr
sudo grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=arch_grub --recheck

    生成grub.cfg,推荐自动化生成:推荐自动化生成:

    sudo grub-mkconfig -o /boot/grub/grub.cfg

  • 10 卸载分区并重启系统

    exit
sudo reboot

简单的配置

至此，安装完成。如果继续折腾的话，可以安装炫酷的窗口管理程序和桌面环境等.Wiki里有很全的指导.
